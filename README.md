### Front-end

- React
- MUI

- React Router
- React Hook Form
- yup
- SWR
- i18

### Back-end

- Express
- Mongoose
- Bcrypt
- JSON Web Token
- Nodemailer
- Multer
- Sharp
- Express Validator
